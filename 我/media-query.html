響應式網頁(Responsive web design,RWD)

  • 1.響應式網頁，讓內容如流水般能適應各種形狀的容器，能在每個裝置尺寸正常瀏覽
  • 2.CSS的媒體查詢(media query)負責響應式主要的工作
  • 3.媒體查詢的斷點(breakpoint)為不同裝置設定通過的位置點，當通過設定的位置點的時候套用該尺寸的樣式
  • 4.使用相對單位設定寬高 大小(rem % vh vw)，字體大小使用 rem ， 容器 width height 使用 %
  • 5.RWD 一定要有這行
